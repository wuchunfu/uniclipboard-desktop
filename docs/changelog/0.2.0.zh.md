## 0.2.0 - 2026-03-09

### Features

- 重新设计剪贴板面板，采用 Raycast 风格的分栏布局
- 在剪贴板条目中展示原始图片而非缩略图，提升预览质量
- 新增运行时主题引擎，提供 12 套配色预设，支持明暗主题平滑切换
- 优化设置页主题预览，采用多色圆点展示配色方案，选择更直观
- 新增剪贴板即时更新 —— 新条目直接出现在顶部，无需整页刷新
- 新增应用内更新日志展示，支持根据用户语言显示双语更新说明
- 新增设置页进入和退出时的圆形展开与收拢动画
- 接收远端剪贴板时仅保留最高优先级的表示格式，减少冗余条目

### Fixes

- 修复 macOS 与 Windows 之间的跨平台剪贴板图片同步问题
- 修复 macOS 图片同步过慢的问题，发送前先将 TIFF 图片转换为 PNG
- 修复页面切换时骨架屏闪烁的问题，在后台刷新期间保留已缓存内容
- 修复开机自启动设置链路，确保后端应用与失败回滚和已保存设置保持一致
- 修复 SQLite 在启动阶段的锁竞争问题，在连接池创建前启用 WAL 模式
- 修复剪贴板面板刷新频率过高导致的频繁重载问题
- 修复 macOS 文本格式下的被动入站剪贴板捕获失效问题
- 修复入站剪贴板回声问题，通过记录远端快照哈希防止重复条目
- 修复条目大小显示错误，改为使用预览表示的大小而非总大小
- 修复因不支持内联数据导致的 "no blob_id" 加载错误
- 修复加解密装饰器中 payload_state 未正确保留的问题
- 修复更新弹窗中更新日志为空的问题，改为从 changelog 文件填充
- 重新生成 macOS Dock 应用图标，改用透明圆角样式

### Refactor

- 简化 DashboardPage 为纯渲染层，将事件管理逻辑提取到专用 Hook
- 移除 `AppPaths` 中未使用的 `keyring_dir` 字段，降低路径漂移风险

### Performance

- 新增 256KB 分块网络传输及进度事件基础设施，为大文件传输做准备
- 优化面板刷新机制：本地捕获使用增量插入，远端同步使用节流重载
