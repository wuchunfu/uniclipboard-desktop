## 0.3.0-alpha.1 - 2026-03-12

### 新功能

- 新增 `uc-observability`，提供日志配置、扁平 JSON 格式化与 tracing subscriber 组合能力。
- 新增 Seq 日志采集模块、启动集成与本地 Docker Compose 支持。
- 在剪贴板捕获、出站同步、入站同步与 blob 暂存流程中新增 `flow_id`/`origin_flow_id` 关联字段和阶段 Span。
- 在 Seq CLEF 事件中注入 `device_id`，支持跨设备链路关联。
- 新增按设备同步设置能力，覆盖领域模型、存储 schema/migration、仓储映射、用例与 Tauri 命令。
- 新增前端 API 与 Redux thunk，并将按设备同步设置接入 `DeviceSettingsPanel`。
- 在 `uc-core` 中新增内容类型分类与过滤，并在设备设置中提供可交互的内容类型开关。

### 修复

- 移除设备设置中的 `allContentTypesDisabled` 警告。
- 修复发送失败时 Seq 批次丢失问题，改为保留批次。
- 在输入通道关闭时，Seq sender 循环可正确退出。
- 将 Seq 信号 JSON 字段改为 Title Case，以匹配格式规范。
- 为入站消息缺失 `origin_flow_id` 的情况增加告警日志。
- 抑制日志配置中 Hyper 连接池的噪声调试日志。
- 完善 mock 与测试实现中的 `sync_settings` 支持，保持行为一致。

### 重构

- 集中化全局 `device_id` 日志上下文处理。
- 重命名全局日志上下文使用的设备 ID 解析器。

### 测试

- 为内容类型分类与过滤新增失败优先（failing-first）测试覆盖。
- 扩展 `DeviceSettingsPanel` 测试，覆盖内容类型开关交互。
- 增加 tracing 路径的 Nyquist 验证测试覆盖。
- 完成第 25 阶段 UAT，4 项通过、1 项跳过。
