## 0.3.0-alpha.7 - 2026-03-16

### Features

- Add per-device sync controls, including content-type toggles.
- Add full keyboard shortcut settings with key recording, conflict detection, and instant updates.
- Add a Storage section in Settings with usage stats, cleanup actions, and quick access to the app data folder.
- Add clearer sync controls with a global paused banner and direct navigation to the Sync settings tab.
- Add cross-device file sync with visible transfer states and failure reasons.
- Add file sync policy controls for enablement, file-size limits, and retention behavior.
- Add a keyboard-first quick panel for faster clipboard history access.
- Add richer link handling for detected URLs, including multi-link entries.
- Add periodic timestamp refresh on the dashboard so relative time labels stay current.
- Add a smoother macOS unlock flow for Keychain permission confirmation.

### Fixes

- Fix Escape key behavior so closing dialogs no longer also closes Settings.
- Fix stale clipboard rows appearing briefly after clearing history.
- Fix retention and storage actions to report errors clearly and recover reliably.
- Fix duplicate or stuck file transfers during cross-device sync.
- Fix duplicate file captures after restore.
- Fix file copy and write reliability by handling clipboard file paths consistently.
- Fix a crash when opening file transfer status tooltips.
