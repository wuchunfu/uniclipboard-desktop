# Plan 42-01: CLI Clipboard Commands

## Goal

Add `clipboard list`, `clipboard get <id>`, and `clipboard clear` subcommands to the `uniclipboard-cli` binary. All three use direct-mode bootstrap (no daemon required): `build_cli_context_with_profile()` + `build_non_gui_runtime()` + `CoreUseCases`. Follow the exact pattern from `commands/devices.rs`.

**Structs to define (all derive Serialize):**

```rust
#[derive(Serialize)]
struct ClipboardEntryRow {
    id: String,
    preview: String,
    content_type: String,
    size_bytes: i64,
    captured_at_ms: i64,
    active_time_ms: i64,
    is_favorited: bool,
}

#[derive(Serialize)]
struct ClipboardListOutput {
    entries: Vec<ClipboardEntryRow>,
    count: usize,
}

#[derive(Serialize)]
struct ClipboardEntryDetail {
    id: String,
    content: String,
    mime_type: Option<String>,
    size_bytes: i64,
    created_at_ms: i64,
    active_time_ms: i64,
}

#[derive(Serialize)]
struct ClipboardClearOutput {
    deleted_count: u64,
    failed_count: usize,
}
```

**Display impls:**

- `ClipboardListOutput`: if empty print "No clipboard entries found.", else print "Clipboard entries: {count}" then each entry as " id={id} {preview} ({content_type}, {size_bytes}B, captured_at={captured_at_ms}, active_time={active_time_ms})"
- `ClipboardEntryDetail`: print id, mime_type (or "unknown"), size_bytes, created_at_ms, active_time_ms, then a blank line, then the full content
- `ClipboardClearOutput`: print "Cleared {deleted_count} clipboard entries." and if failed_count > 0 also print " {failed_count} entries failed to delete."

**run_list signature:**

```rust
pub async fn run_list(json: bool, verbose: bool, limit: usize, offset: usize) -> i32
```

Bootstrap pattern (identical to devices.rs):

1. Build profile: `let profile = if verbose { Some(uc_observability::LogProfile::Dev) } else { Some(uc_observability::LogProfile::Cli) };`
2. `uc_bootstrap::build_cli_context_with_profile(profile)`
3. `uc_bootstrap::get_storage_paths(&ctx.config)`
4. `uc_bootstrap::build_non_gui_runtime(ctx.deps, storage_paths)`
5. `let usecases = uc_app::usecases::CoreUseCases::new(&runtime)`
6. `match usecases.list_entry_projections().execute(limit, offset).await` — on `Err(e)`, `eprintln!("Error: failed to list clipboard entries: {}", e)` and return `EXIT_ERROR`
7. Map each `EntryProjectionDto` to `ClipboardEntryRow` (id, preview, content_type, size_bytes, captured_at as captured_at_ms, active_time as active_time_ms, is_favorited)
8. Build `ClipboardListOutput` from the mapped rows and call `output::print_result(&result, json)`

**run_get signature:**

```rust
pub async fn run_get(json: bool, verbose: bool, id: String) -> i32
```

1. Same bootstrap steps 1-5
2. Construct `entry_id` via `let entry_id = uc_core::ids::EntryId::from_str(&id);` — `EntryId::from_str` is infallible (no error branch needed)
3. `match usecases.get_entry_detail().execute(&entry_id).await` — on `Err(e)`, `eprintln!("Error: {}", e)` and return `EXIT_ERROR` (use case returns Err for unknown IDs and non-text entries such as images or files)
4. Map `EntryDetailResult` to `ClipboardEntryDetail`
5. `output::print_result(&result, json)`

**run_clear signature:**

```rust
pub async fn run_clear(json: bool, verbose: bool) -> i32
```

1. Same bootstrap steps 1-5
2. `usecases.clear_clipboard_history().execute().await`
3. Map `ClearHistoryResult` to `ClipboardClearOutput { deleted_count, failed_count: failed_entries.len() }`
4. `output::print_result(&result, json)`

**Imports needed:**

```rust
use serde::Serialize;

use crate::output;
// uc_bootstrap and uc_app are referenced via fully-qualified paths
```

For `run_get`, also import:

```rust

```

Add a new enum after `Commands`:

```rust
#[derive(Subcommand)]
enum ClipboardCommands {
    /// List clipboard history entries
    List {
        /// Maximum number of entries to return (default: 50)
        #[arg(long, default_value_t = 50)]
        limit: usize,
        /// Number of entries to skip (default: 0)
        #[arg(long, default_value_t = 0)]
        offset: usize,
    },
    /// Get full detail for a single clipboard entry
    Get {
        /// Entry ID
        id: String,
    },
    /// Clear all clipboard history
    Clear,
}
```

Add dispatch arm in `rt.block_on(async { match cli.command { ... } })`:

```rust
Commands::Clipboard { subcommand } => match subcommand {
    ClipboardCommands::List { limit, offset } => {
        commands::clipboard::run_list(cli.json, cli.verbose, limit, offset).await
    }
    ClipboardCommands::Get { id } => {
        commands::clipboard::run_get(cli.json, cli.verbose, id).await
    }
    ClipboardCommands::Clear => {
        commands::clipboard::run_clear(cli.json, cli.verbose).await
    }
},
```
