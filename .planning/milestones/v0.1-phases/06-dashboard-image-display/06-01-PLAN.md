<objective>
Fix dashboard image display by converting uc:// protocol URLs to platform-correct format using Tauri's convertFileSrc API.

Purpose: Images are captured successfully (Phase 5) but not visible in the dashboard because uc:// URLs are used directly in img src and fetch calls. Tauri 2 custom URI scheme protocols use different URL formats per platform (macOS/Linux: uc://localhost/{path}, Windows: http://uc.localhost/{path}). The fix is to create a resolveUcUrl helper using convertFileSrc and apply it everywhere uc:// URLs are consumed.

Output: Working image display in dashboard on all platforms.
</objective>
