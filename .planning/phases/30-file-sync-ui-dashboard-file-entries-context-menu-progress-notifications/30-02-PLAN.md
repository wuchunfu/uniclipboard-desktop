1. Create `src/store/slices/fileTransferSlice.ts`:
   ```typescript
   import { createSlice, PayloadAction } from '@reduxjs/toolkit'

   export interface TransferProgressInfo {
     transferId: string
     entryId: string | null // linked to clipboard entry, set when known
     peerId: string
     direction: 'Sending' | 'Receiving'
     chunksCompleted: number
     totalChunks: number
     bytesTransferred: number
     totalBytes: number | null
     status: 'active' | 'completed' | 'failed'
     startedAt: number
     updatedAt: number
   }

   interface FileTransferState {
     activeTransfers: Record<string, TransferProgressInfo>
     // Map entry_id -> transfer_id for linking UI items to transfers
     entryTransferMap: Record<string, string>
   }

   const initialState: FileTransferState = {
     activeTransfers: {},
     entryTransferMap: {},
   }
   ```
   - Reducers:
     - `updateTransferProgress(state, action: PayloadAction<{ transferId, peerId, direction, chunksCompleted, totalChunks, bytesTransferred, totalBytes }>)`:
       - Upsert into `activeTransfers` keyed by `transferId`
       - Set `status: 'active'`, `updatedAt: Date.now()`
       - If `chunksCompleted === totalChunks`, set `status: 'completed'`
     - `linkTransferToEntry(state, action: PayloadAction<{ transferId: string, entryId: string }>)`:
       - Set `activeTransfers[transferId].entryId = entryId`
       - Set `entryTransferMap[entryId] = transferId`
     - `markTransferFailed(state, action: PayloadAction<{ transferId: string }>)`:
       - Set `activeTransfers[transferId].status = 'failed'`
     - `clearCompletedTransfers(state)`:
       - Remove transfers with `status === 'completed'` from both maps
     - `removeTransfer(state, action: PayloadAction<string>)`:
       - Remove by transferId from both maps
   - Export selectors:
     - `selectTransferByEntryId(state, entryId)` -> TransferProgressInfo | undefined
     - `selectActiveTransfers(state)` -> TransferProgressInfo[]
     - `selectIsEntryTransferring(state, entryId)` -> boolean

2. In `src/store/index.ts`:
   - Import `fileTransferReducer` from `./slices/fileTransferSlice`
   - Add `fileTransfer: fileTransferReducer` to the reducer map

3. Create `src/hooks/useTransferProgress.ts`:
   - Listen to `"transfer://progress"` Tauri events
   - On each event, dispatch `updateTransferProgress` with the payload data
   - Auto-clear completed transfers after a delay (e.g., 3 seconds) using `setTimeout` + `removeTransfer`
   - Cleanup: unlisten on unmount, clear timeouts
   - Usage: Call this hook once in DashboardPage (or a top-level component)
   - Pattern: Follow the same structure as `useClipboardEvents` for consistency (useEffect with listen, cancelled flag, cleanup)
