   ```typescript
   interface TransferState {
     transferId: string;
     peerId: string;
     direction: "Sending" | "Receiving";
     chunksCompleted: number;
     totalChunks: number;
     bytesTransferred: number;
     totalBytes: number;
     updatedAt: number; // Date.now() for auto-cleanup
   }

interface TransferSliceState {
activeTransfers: Record<string, TransferState>;
}

```
- `updateTransferProgress` reducer: upsert by transferId, set updatedAt to Date.now(). If chunksCompleted === totalChunks, mark as complete.
- `clearTransfer` reducer: remove by transferId
- `clearStaleTransfers` reducer: remove entries where updatedAt is > 5 seconds old AND 100% complete
- Export selectors: `selectActiveTransfers` returns array of non-complete transfers sorted by updatedAt desc

2. In `src/store/index.ts`:
- Import and add `transfer: transferSlice.reducer` to the store configuration

3. Create `src/hooks/useTransferProgress.ts`:
- Subscribe to `"transfer://progress"` Tauri event using `listen()`
- On each event, dispatch `updateTransferProgress(event.payload)`
- Set up a 2-second interval to dispatch `clearStaleTransfers()` for auto-cleanup of completed transfers
- Return cleanup function that calls unlisten and clears interval
- This hook should be called once from DashboardPage (or a layout component)
