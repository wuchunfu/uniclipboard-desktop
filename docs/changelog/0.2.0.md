## 0.2.0 - 2026-03-09

### Features

- Redesign clipboard dashboard with Raycast-style split view layout
- Display original images instead of thumbnails in clipboard items for better preview quality
- Add runtime theme engine with 12 color presets and smooth light/dark theme transitions
- Add richer theme preset previews in Settings with multi-dot color swatches
- Add instant clipboard updates — new items appear at the top without full page reload
- Add in-app release notes rendering with bilingual update notes support
- Add circular reveal and collapse animations for Settings page navigation
- Keep only highest-priority representation in inbound clipboard sync to reduce clutter

### Fixes

- Fix cross-platform clipboard image sync between macOS and Windows
- Fix slow macOS image sync by converting clipboard TIFF to PNG before transfer
- Fix page navigation skeleton flashes so cached content stays visible during background refresh
- Fix autostart settings so backend application and rollback stay consistent with the saved setting
- Fix SQLite startup lock contention by enabling WAL mode before pool creation
- Fix clipboard dashboard refresh throttling to avoid excessive reloads
- Fix passive inbound clipboard capture for macOS text formats
- Fix inbound clipboard echo by recording remote snapshot hash to prevent duplicates
- Fix entry display size showing total size instead of preview representation size
- Fix "no blob_id" errors by supporting inline data in entry resource loading
- Fix payload_state preservation in encryption/decryption decorators
- Fix empty update release notes by populating the updater manifest from changelog files
- Regenerate app icons with transparent rounded corners for macOS Dock

### Refactor

- Simplify DashboardPage to a thin render layer with dedicated event management hooks
- Remove unused `keyring_dir` from `AppPaths` struct to reduce path drift risk

### Performance

- Add chunked 256KB network transfer with progress event foundations for large payloads
- Optimize dashboard refresh with incremental prepend for local captures and throttled reload for remote sync
