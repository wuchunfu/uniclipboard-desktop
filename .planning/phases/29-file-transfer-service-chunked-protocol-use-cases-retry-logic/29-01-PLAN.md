   - `write_file_frame(writer, msg_type, payload)`: writes 1-byte type tag + length-prefixed payload

   - Max frame size: 256KB + 1KB overhead (for announce/complete metadata)
   - Reuse length-prefixed pattern from pairing_stream framing

3. **Create `src-tauri/crates/uc-platform/src/adapters/file_transfer/protocol.rs`**:
   - Constants: `CHUNK_SIZE: usize = 256 * 1024` (256KB)
   - Structs:
     ```rust
     pub struct FileAnnounce {
         pub transfer_id: String,
         pub filename: String,
         pub file_size: u64,
         pub blake3_hash: String,
         pub batch_id: Option<String>,
         pub batch_total: Option<u32>,
     }

     pub struct FileAcceptance {
         pub transfer_id: String,
         pub accepted: bool,
         pub reason: Option<String>,
     }

     pub struct FileChunkHeader {
         pub chunk_index: u32,
         pub chunk_size: u32,
     }

     pub struct FileComplete {
         pub transfer_id: String,
         pub blake3_hash: String,
         pub total_chunks: u32,
     }
     ```
   - All structs derive Serialize/Deserialize for JSON encoding within frames
   - `send_file_chunked(writer, file_path, transfer_id, progress_tx)`: reads file in CHUNK_SIZE chunks, computes Blake3 hash incrementally, sends each chunk via framing, emits TransferProgress per chunk, sends FileComplete at end
   - `receive_file_chunked(reader, announce, cache_dir, progress_tx)`: creates temp file (.tmp suffix), writes chunks as received, computes Blake3 incrementally, verifies hash on FileComplete, atomic rename, returns final path
   - `compute_blake3_hash(file_path)`: full-file Blake3 hash for pre-send computation
   - Atomic write pattern: write to `{cache_dir}/{transfer_id}.tmp`, verify hash, rename to `{cache_dir}/{transfer_id}_{filename}`
   - Unix permissions: `#[cfg(unix)]` set 0o600 on temp file after creation

   Follow PairingStreamService pattern:

   ```rust
   pub const MAX_FILE_TRANSFER_CONCURRENCY: usize = 8;
   const PER_PEER_FILE_CONCURRENCY: usize = 2;

   struct FileTransferServiceInner {

       config: FileTransferConfig,
   }
   ```

   Methods:

   - `acquire_permits(&self, peer_id)`: global + per-peer semaphore (same as PairingStreamService)
