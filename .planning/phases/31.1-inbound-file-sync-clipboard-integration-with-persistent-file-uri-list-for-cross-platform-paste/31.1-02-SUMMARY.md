## Decisions Made

- Extension-based icon map uses constant `Record<string, React.ElementType>` with map lookup (`FILE_EXT_ICON_MAP[ext]`) rather than function call returning component, to satisfy ESLint `react-hooks/static-components` rule
- Lazy stale detection approach: no startup file scanning; staleness only discovered when user tries to Copy and backend returns error
- Delete cascade parses `inline_data` field of text/uri-list representations to extract file:// URIs and delete cache files
- `ClearClipboardHistory` also updated to pass `representation_repo` so bulk clear also triggers cache file cleanup
