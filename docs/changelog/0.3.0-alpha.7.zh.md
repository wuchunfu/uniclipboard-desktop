## 0.3.0-alpha.7 - 2026-03-16

### Features

- 新增按设备同步控制，并支持内容类型开关。
- 新增完整的快捷键设置，支持按键录制、冲突检测与即时更新。
- 新增设置页存储管理，支持查看空间占用、执行清理操作并快速打开应用数据目录。
- 新增更清晰的同步控制：全局暂停横幅与直达同步设置页签的导航入口。
- 新增跨设备文件同步，并可查看传输状态与失败原因。
- 新增文件同步策略设置，支持开关、文件大小限制与保留策略。
- 新增键盘优先的快捷面板，可更快访问剪贴板历史。
- 新增更丰富的链接识别与展示，支持多链接条目。
- 新增仪表盘时间戳定时刷新，确保相对时间显示保持准确。
- 新增更顺畅的 macOS Keychain 权限确认解锁流程。

### Fixes

- 修复按下 Escape 关闭弹窗时连带关闭设置页的问题。
- 修复清空历史后短暂出现过期剪贴板条目的问题。
- 修复保留策略与存储操作的错误反馈与恢复流程，使失败提示更明确。
- 修复跨设备同步中传输记录重复或卡住的问题。
- 修复恢复后文件被重复采集的问题。
- 修复文件复制与写入的路径处理问题，提升稳定性。
- 修复打开文件传输状态提示时可能发生的崩溃。
