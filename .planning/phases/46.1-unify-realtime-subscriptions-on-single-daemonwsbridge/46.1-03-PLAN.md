- `pub struct DaemonWsBridge`

- becomes the only realtime startup path launched from `start_background_tasks(...)`

Modify `setup_pairing_bridge.rs` only enough to accept the shared setup event hub and stop owning transport lifecycle. Do not delete the legacy websocket code in this plan; Phase 46.1-05 removes it after the frontend cutover is proven.
