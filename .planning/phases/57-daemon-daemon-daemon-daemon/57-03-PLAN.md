<objective>
Add write-back loop prevention to the daemon clipboard watcher by integrating ClipboardChangeOriginPort into DaemonClipboardChangeHandler.

Purpose: Per D-09, the daemon will also write to the OS clipboard (inbound sync from remote peers). When daemon writes to the clipboard, the ClipboardWatcher will detect the change and re-capture it. The ClipboardChangeOriginPort prevents this by marking writes with RemotePush origin and checking origin on capture. This plan wires the origin port into the daemon's clipboard handler so it's ready when inbound sync is added.

Output: DaemonClipboardChangeHandler checks origin via ClipboardChangeOriginPort, shared instance ready for inbound sync to use.
</objective>

Key flow for inbound sync write-back prevention:

1. SyncInboundClipboardUseCase receives remote clipboard content
2. Calls clipboard_change_origin.remember_remote_snapshot_hash(hash, TTL) BEFORE write
3. Writes to OS clipboard via local_clipboard.write_snapshot()
4. Calls clipboard_change_origin.set_next_origin(RemotePush, TTL) AFTER write
5. ClipboardWatcher detects the change, reads snapshot
6. DaemonClipboardChangeHandler calls consume_origin_for_snapshot_or_default(hash, LocalCapture)
7. Origin port recognizes the hash -> returns RemotePush instead of LocalCapture
8. CaptureClipboardUseCase tags entry with RemotePush origin
9. OutboundSyncPlanner sees RemotePush and does NOT re-sync to other peers

    async fn on_clipboard_changed(&self, snapshot: SystemClipboardSnapshot) -> Result<()> {
        // 1. Compute snapshot hash for origin check
        let snapshot_hash = snapshot.snapshot_hash().to_string();

        // 2. Check if this clipboard change was caused by our own write (inbound sync)
        let origin = self.clipboard_change_origin
            .consume_origin_for_snapshot_or_default(
                &snapshot_hash,
                ClipboardChangeOrigin::LocalCapture,
            )
            .await;

        // 3. Capture with the correct origin
        let usecase = CaptureClipboardUseCase::new(/* deps from runtime */);
        match usecase.execute_with_origin(snapshot, origin).await {
            Ok(Some(entry_id)) => {
                // 4. Determine origin kind for WS event
                let origin_str = match origin {
                    ClipboardChangeOrigin::LocalCapture | ClipboardChangeOrigin::LocalRestore => "local",
                    ClipboardChangeOrigin::RemotePush => "remote",
                };
                // 5. Broadcast WS event
                let _ = self.event_tx.send(DaemonWsEvent { /* ... */ });
            }
            Ok(None) => { /* dedup at use case level, no action */ }
            Err(e) => { warn!(error = %e, "clipboard capture failed"); }
        }
        Ok(())
    }
