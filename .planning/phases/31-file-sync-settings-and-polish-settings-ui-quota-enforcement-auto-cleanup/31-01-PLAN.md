   **Update useEffect** to sync file sync state from settings:
   ```typescript
   // Inside existing useEffect where setting is synced:
   setFileSyncEnabled(setting.sync.file_sync_enabled ?? true)
   setSmallFileThreshold(setting.sync.small_file_threshold_mb ?? 10)
   setMaxFileSizeLimit(setting.sync.max_file_size_limit_mb ?? 5120)
   setCacheQuota(setting.sync.file_cache_quota_per_device_mb ?? 500)
   setRetentionHours(setting.sync.file_retention_hours ?? 24)
   setFileAutoCleanup(setting.sync.file_auto_cleanup ?? true)
   ```

   **Add handlers** for each setting:

   a. `handleFileSyncEnabledChange(checked: boolean)`:
      - Set local state
      - Call `updateSyncSetting({ file_sync_enabled: checked })`

   b. `handleSmallFileThresholdChange(e)`:
      - Validate: must be positive integer, range 1-1000
      - Cross-validate: must be less than maxFileSizeLimit
      - On valid: call `updateSyncSetting({ small_file_threshold_mb: value })`
      - Follow existing `handleMaxFileSizeChange` pattern for validation and error display

   c. `handleMaxFileSizeLimitChange(e)`:
      - Validate: must be positive integer, range 1-10240 (10GB in MB)
      - On valid: call `updateSyncSetting({ max_file_size_limit_mb: value })`

   d. `handleCacheQuotaChange(e)`:
      - Validate: must be positive integer, range 50-10240
      - On valid: call `updateSyncSetting({ file_cache_quota_per_device_mb: value })`

   e. `handleRetentionHoursChange(e)`:
      - Validate: must be positive integer, range 1-720
      - On valid: call `updateSyncSetting({ file_retention_hours: value })`

   f. `handleFileAutoCleanupChange(checked: boolean)`:
      - Set local state
      - Call `updateSyncSetting({ file_auto_cleanup: checked })`
