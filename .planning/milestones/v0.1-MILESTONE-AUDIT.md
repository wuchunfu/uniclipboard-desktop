## E2E Flow Verification

### Flow 1: V2 Clipboard Sync (Device A → Device B) — VERIFIED

```
Device A: SystemClipboard read_snapshot()
  → SyncOutboundClipboardUseCase: pack all reps into ClipboardMultiRepPayloadV2
  → ChunkedEncoder::encode_to (bounded memory ~CHUNK_SIZE×2)
  → ProtocolMessage::frame_to_bytes(Some(&encrypted_content))
  → ClipboardTransportPort::send_clipboard (libp2p stream)
  → [wire: 4-byte JSON len | JSON header | raw V2 binary]
Device B: Two-segment framing reader
  → spawn_blocking { SyncIoBridge → ChunkedDecoder::decode_from } (~1x chunk memory)
  → clipboard_tx.send((message, Some(plaintext)))
  → run_clipboard_receive_loop: execute_with_outcome(message, Some(plaintext))
  → SyncInboundClipboardUseCase::apply_v2_inbound
  → Pre-decoded fast path (skip master key fetch + ChunkedDecoder)
  → select_highest_priority_repr (image > html > rtf > plain > other)
  → local_clipboard.write_snapshot()
```

### Flow 2: V1 Backward Compatibility (Old Device A → New Device B) — VERIFIED

```
Old Device A: Sends ProtocolMessage JSON (payload_version absent)
  → [wire: 4-byte JSON len | JSON]
Device B: Two-segment framing reader
  → from_bytes() → payload_version = V1 (serde default)
  → ProcessedMessage::Standard → handle_standard_message
  → clipboard_tx.send((message, None))
  → execute_with_outcome(message, None) → apply_v1_inbound
  → decrypt_blob + parse ClipboardTextPayloadV1
  → write text/plain to OS clipboard
```
