<objective>
Remove all daemon-duplicated background sync loops from uc-tauri wiring.rs.

Purpose: The daemon now owns clipboard inbound sync (InboundClipboardSyncWorker), network event handling (FileSyncOrchestratorWorker, PeerMonitor, PeerDiscoveryWorker), startup reconciliation, and timeout sweeps. The wiring.rs copies of these loops are dead weight that only execute in standalone Full mode — which the project no longer supports as primary architecture.

Output: A leaner wiring.rs with only GUI-owned storage tasks (spool scanner, spooler, blob worker, spool janitor), DaemonWsBridge realtime runtime, and file cache cleanup. All daemon-duplicated functions, constants, helpers, and their associated imports are removed.
</objective>
