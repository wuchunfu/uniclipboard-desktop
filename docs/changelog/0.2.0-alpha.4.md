## 0.2.0-alpha.4 - 2026-03-09

### Features

- Add a runtime theme engine with 12 presets and smooth light/dark theme transitions
- Add richer theme preset previews in Settings for faster visual selection
- Improve chunk transfer infrastructure with resume and progress auditing foundations

### Fixes

- Fix cross-platform clipboard image sync between macOS and Windows
- Fix autostart settings so backend application and rollback stay consistent with the saved setting
- Fix SQLite startup lock contention by enabling WAL mode before pool creation

### Refactor

- Remove the unused `keyring_dir` field from `AppPaths` to reduce path drift risk

### Docs

- Add a markdown changelog template for future releases
