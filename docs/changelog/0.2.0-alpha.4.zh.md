## 0.2.0-alpha.4 - 2026-03-09

### Features

- 新增运行时主题引擎，提供 12 套主题预设，并支持平滑的明暗主题切换
- 优化设置页中的主题预览，让主题配色选择更直观
- 增强分块传输基础能力，改进断点恢复与传输进度审计相关基础设施

### Fixes

- 修复 macOS 与 Windows 之间的跨平台剪贴板图片同步问题
- 修复开机自启动设置链路，确保后端应用与失败回滚和已保存设置保持一致
- 修复 SQLite 在启动阶段的锁竞争问题，通过在连接池创建前启用 WAL 模式提升稳定性

### Refactor

- 移除 `AppPaths` 中未使用的 `keyring_dir` 字段，降低路径漂移风险

### Docs

- 新增用于后续版本发布的 Markdown changelog 模板
