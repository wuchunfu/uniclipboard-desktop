## Current Problem

Two Tauri commands aggregate peers in the command layer:

1. `get_p2p_peers` (lines ~142-196 in pairing.rs) - aggregates discovered + connected
2. `get_paired_peers_with_status` (lines ~207-272 in pairing.rs) - aggregates paired + discovered + connected

This cross-use-case aggregation in the command layer:

1. Is not reusable from CLI
2. Duplicates the aggregation logic in two commands
3. Not testable as a unit
4. CLI devices.rs only uses list_paired_devices (incomplete view)

## Target Solution

Create a comprehensive `GetP2pPeersSnapshot` use case in uc-app that:

1. Lists discovered peers via PeerDirectoryPort
2. Lists connected peers via PeerDirectoryPort
3. Lists paired devices via PairedDeviceRepositoryPort
4. Merges all three into a unified snapshot with pairing_state and identity_fingerprint

Both Tauri commands and CLI will use this shared use case.

use uc_core::network::{ConnectedPeer, DiscoveredPeer};
use uc_core::ports::paired_device_repository::PairedDevice;

pub struct P2pPeerSnapshot {
    pub peer_id: String,
    pub device_name: Option<String>,
    pub addresses: Vec<String>,
    pub is_paired: bool,
    pub is_connected: bool,
    pub pairing_state: String,      // PRESERVED per FINDING-4
    pub identity_fingerprint: String, // PRESERVED per FINDING-4
}

pub struct GetP2pPeersSnapshot {
    // FIXED: Both ports required per FINDING-2
    peer_dir: Arc<dyn PeerDirectoryPort>,
    paired_repo: Arc<dyn PairedDeviceRepositoryPort>,
}

impl GetP2pPeersSnapshot {
    pub fn new(
        peer_dir: Arc<dyn PeerDirectoryPort>,
        paired_repo: Arc<dyn PairedDeviceRepositoryPort>,
    ) -> Self {
        Self { peer_dir, paired_repo }
    }

    pub async fn execute(&self) -> anyhow::Result<Vec<P2pPeerSnapshot>> {
        // 1. List discovered peers
        let discovered = self.peer_dir.list_discovered_peers()?;

        // 2. List connected peers
        let connected = self.peer_dir.list_connected_peers()?;
        let connected_ids: HashSet<_> = connected.iter().map(|p| p.peer_id.clone()).collect();

        // 3. List paired devices
        let paired = self.paired_repo.list_paired_devices()?;
        let paired_map: HashMap<_, _> = paired.iter()
            .map(|p| (p.peer_id.clone(), p))
            .collect();

        // 4. Merge into snapshot
        let mut snapshots = Vec::new();

        // Add discovered peers
        for peer in discovered {
            let paired_dev = paired_map.get(&peer.peer_id);
            snapshots.push(P2pPeerSnapshot {
                peer_id: peer.peer_id.clone(),
                device_name: paired_dev.and_then(|p| p.device_name.clone())
                    .or(peer.device_name),
                addresses: peer.addresses.iter().map(|a| a.to_string()).collect(),
                is_paired: paired_dev.is_some(),
                is_connected: connected_ids.contains(&peer.peer_id),
                pairing_state: paired_dev
                    .map(|p| format!("{:?}", p.pairing_state))
                    .unwrap_or_else(|| "NotPaired".to_string()),
                identity_fingerprint: paired_dev
                    .map(|p| p.identity_fingerprint.clone())
                    .unwrap_or_default(),
            });
        }

        // Add paired but not discovered
        for (peer_id, dev) in paired_map {
            if !connected_ids.contains(peer_id) {
                snapshots.push(P2pPeerSnapshot {
                    peer_id: peer_id.clone(),
                    device_name: dev.device_name.clone(),
                    addresses: vec![],
                    is_paired: true,
                    is_connected: false,
                    pairing_state: format!("{:?}", dev.pairing_state),
                    identity_fingerprint: dev.identity_fingerprint.clone(),
                });
            }
        }

        Ok(snapshots)
    }
}
```

#[tauri::command]
pub async fn get_paired_peers_with_status(
    runtime: State<'_, AppRuntime>,
) -> Result<Vec<PairedPeerInfo>, String> {
    let snapshot = runtime.usecases().get_p2p_peers_snapshot().execute()
        .await
        .map_err(|e| e.to_string())?;

    let peers: Vec<PairedPeerInfo> = snapshot
        .into_iter()
        .filter(|p| p.is_paired)  // Only paired
        .map(|p| PairedPeerInfo {
            peer_id: p.peer_id,
            device_name: p.device_name.unwrap_or_default(),
            pairing_state: p.pairing_state,
            identity_fingerprint: p.identity_fingerprint,
            is_connected: p.is_connected,
        })
        .collect();

    Ok(peers)
}
```
