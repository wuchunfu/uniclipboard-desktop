Tauri commands available:
```
get_device_sync_settings(peerId: string) -> SyncSettings
update_device_sync_settings(peerId: string, settings: SyncSettings | null) -> void
```

SyncSettings shape (from Rust, camelCase via serde rename):
```typescript
interface SyncSettings {
  auto_sync: boolean         // Note: Rust uses snake_case serde default
  sync_frequency: 'realtime' | 'interval'
  content_types: {
    text: boolean
    image: boolean
    link: boolean
    file: boolean
    code_snippet: boolean
    rich_text: boolean
  }
  max_file_size_mb: number
}
```

Note: Rust SyncSettings uses `#[serde(rename_all = "snake_case")]` on SyncFrequency but NOT on SyncSettings itself -- so field names remain snake_case in JSON. The frontend must use snake_case field names when sending/receiving. Verify the exact serialization format when implementing.
