- Multi-file merging: "Syncing N files to [device]" -> "All N files synced"

- Clipboard race: if user copies during transfer, auto-write cancelled

   ```typescript
   const BATCH_WINDOW_MS = 500

   export function useFileSyncNotifications() {
     const { t } = useTranslation()
     const activeTransfers = useAppSelector(selectActiveTransfers)
     const prevTransfersRef = useRef<Record<string, TransferProgressInfo['status']>>({})
     const batchTimerRef = useRef<number | null>(null)
     const pendingStartBatch = useRef<{ peerId: string; count: number }[]>([])
     const pendingCompleteBatch = useRef<{ peerId: string; count: number }[]>([])
   ```

   - **Batch start notifications**: When new transfers appear (not in prevTransfersRef), add to pendingStartBatch. After BATCH_WINDOW_MS, emit single notification:
     - 1 file: "File syncing to [device]"
     - N files: "Syncing N files to [device]"
   - **Batch completion notifications**: When transfers transition to 'completed', add to pendingCompleteBatch. After BATCH_WINDOW_MS:
     - 1 file: "Sync complete"
     - N files: "All N files synced"
   - **Error notifications**: When transfer status becomes 'failed', emit immediately (no batching):
     - "File sync failed: [reason]"
   - **Permission handling**: Check permission on mount, request if not granted, skip notifications if denied

   - Use `useEffect` watching `activeTransfers` to detect state transitions
   - Update `prevTransfersRef` after processing

     ```typescript
     markTransferFailed(state, action: PayloadAction<{ transferId: string; error?: string }>) {
       const transfer = state.activeTransfers[action.payload.transferId]
       if (transfer) {
         transfer.status = 'failed'
         transfer.errorMessage = action.payload.error
       }
     }
     ```
   - Add `cancelClipboardWrite` reducer:
     ```typescript
     cancelClipboardWrite(state, action: PayloadAction<string>) {
       // Cancel auto-clipboard-write for all active transfers when user copies something new
       for (const transfer of Object.values(state.activeTransfers)) {
         if (transfer.status === 'active') {
           transfer.clipboardWriteCancelled = true
         }
       }
     }
     ```
