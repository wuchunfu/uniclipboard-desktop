## 0.3.0 - 2026-03-16

### Features

- Add cross-platform file sync between paired devices with real-time progress tracking.
- Add file transfer status that persists across restarts (pending, transferring, completed, failed).
- Add file sync settings: enable/disable, max file size, retention period, and automatic cleanup.
- Add file context menu with Open, Show in Finder/Explorer, Copy, and Delete actions.
- Add quick panel for instant clipboard history access via a system-wide shortcut.
- Add keyboard shortcuts customization in Settings.
- Add per-device sync settings with auto-sync toggle and content-type filters (text, image, link, file).
- Add global sync toggle to pause all clipboard synchronization.
- Add link content type with URL detection and multi-URL rendering.
- Add storage management in Settings: view usage, clear cache, clear history, and open data directory.
- Add macOS Keychain confirmation dialog during unlock flow.
- Improve setup flow with draggable header, back button, and device discovery showing real device names.
- Add dashboard layout persistence and auto-refreshing timestamps.

### Fixes

- Fix quick panel dismiss and focus issues on Windows.
- Fix quick panel shortcut inconsistencies across platforms.
- Fix pairing cancel/reject leaving the pairing flow stuck.
- Fix sponsor unable to re-authorize access for new joiners.
- Fix duplicate entries when syncing files across devices.
- Fix file transfer status stuck at pending.
- Fix user files being deleted when clearing clipboard entries.
- Fix stale items flashing on dashboard after clearing history.
- Fix Escape key in Settings dialogs closing the entire page.
- Fix setup flow step indicator order and device name display.
