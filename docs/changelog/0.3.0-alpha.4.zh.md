## 0.3.0-alpha.4 - 2026-03-13

### Features

- 新增完整的键盘快捷键设置，支持交互式按键录制、冲突检测与实时生效。
- 在设置页新增存储管理模块，支持查看存储统计、清理缓存、清空历史与打开数据目录。
- 将清空历史的浏览器确认框替换为应用内对话框，并支持键盘快捷确认。
- 在设备页新增全局同步暂停提示，并在全局同步关闭时禁用设备同步控制项。
- 新增从其他流程直接打开设置页 Sync 标签的导航能力。
- 在仪表盘新增周期性时间刷新，确保相对时间显示持续更新。
- 引入新的排版系统，更新字体族、字号标尺与 CJK 字体支持。
- 在同步过滤策略中强制执行全局自动同步规则，关闭后阻止出站目标。

### Fixes

- 修复从设置页清空历史后旧剪贴板条目短暂闪现的问题，清理成功后立即重置 Redux 缓存。
- 修复保留策略更新未等待后端结果的问题，并在失败时回滚 UI 状态。
- 修复保留策略的竞态问题，改为使用最新的乐观规则快照。
- 修复存储统计加载失败被静默吞掉的问题，改为显示明确错误和重试入口。
- 修复清空历史在大数据量下分页参数非法导致失败的问题。
- 修复打开数据目录时依赖字符串匹配判错的脆弱逻辑，改为类型化下转判断。
- 修复缓存目录大小统计将 IO 错误误报为 0 的问题，改为向上透传错误。
- 修复存储路径解析忽略配置路径与 profile 后缀的问题。
- 修复 `DialogFooter` 底部间距异常，去除多余空隙并保持对齐。
- 修复设置页路由状态更新方式，改用 React Router 导航而非直接替换 history。
- 修复设置开关初始化时的闪烁问题，改为从已加载的上下文状态初始化本地状态。
- 修复快捷键与 i18n 的边界问题，移除硬编码文案并完善更新失败处理。

### Refactor

- 将存储相关命令处理器重构为独立用例，统一到六边形架构的用例层。
- 将清理缓存的文件系统操作重构到 `CacheFsPort` 适配器边界后。
- 将清空剪贴板历史的命令流程重构为独立用例，并返回结构化结果。
- 新增 `src/api/storage.ts` 类型化存储 IPC 包装层，集中前端存储调用。

### Tests

- 为 `GetStorageStats` 与 `OpenDataDirectory` 用例新增单元测试。
- 新增全局自动同步策略测试，覆盖覆盖规则、回退、恢复与不可变性场景。
- 新增相对时间自动刷新行为测试（基于 fake timer）。

### Docs

- 新增并补全全局同步主开关与快捷键设置相关的规划和验证文档。
- 新增清空历史后仪表盘闪烁问题的调试分析记录。
- 完善相对时间刷新及相关执行跟踪的阶段总结与计划记录。
