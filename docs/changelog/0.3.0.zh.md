## 0.3.0 - 2026-03-16

### Features

- 新增跨平台文件同步，配对设备间共享文件，支持实时进度追踪。
- 新增持久化文件传输状态（等待/传输中/已完成/失败），重启后保留。
- 新增文件同步设置：开关、最大文件大小、保留周期与自动清理。
- 新增文件右键菜单，支持打开、在 Finder/资源管理器中显示、复制和删除。
- 新增快捷面板，通过系统级快捷键即时访问剪贴板历史。
- 新增快捷键自定义设置。
- 新增逐设备同步设置，支持自动同步开关和内容类型过滤（文本、图片、链接、文件）。
- 新增全局同步总开关，一键暂停所有剪贴板同步。
- 新增链接内容类型，自动识别 URL 并支持多链接展示。
- 新增存储管理：查看用量、清除缓存、清除历史记录和打开数据目录。
- 新增 macOS 解锁时钥匙串确认弹窗。
- 改进设置引导流程，支持拖拽标题栏、返回按钮，设备发现显示真实设备名称。
- 新增仪表盘布局记忆和时间戳自动刷新。

### Fixes

- 修复 Windows 上快捷面板消失和焦点丢失问题。
- 修复快捷面板快捷键在多平台上不一致。
- 修复配对取消/拒绝后流程卡住。
- 修复发起者无法为后续加入者重新授权。
- 修复跨设备文件同步时出现重复条目。
- 修复文件传输状态卡在"等待中"。
- 修复删除剪贴板条目时误删用户文件。
- 修复清除历史后仪表盘闪烁残留数据。
- 修复设置页对话框 Escape 键误关整个页面。
- 修复设置引导中步骤顺序和设备名称显示。
